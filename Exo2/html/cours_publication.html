Quelques projets réalisés par les étudiants gradués

Vous trouverez ci-après les présentations de quelques projets réalisés dans le cadre du cours par les étudiants. Ces projets mobilisent une bonne partie, si ce n'est l'ensemble des connaissances acquises durant le cours mais vont également un peu plus loin en leur permettant d'investiguer des outils, méthodes, formats ou protocoles pas ou partiellement abordés dans le cours :

  • 2012
    • Étude comparative de serveurs open source conformes WPS : GeoServer, ZooWPS, Deegree et 52 North (pptx)
      Kaboro Samassé
  • 2009
    • Étude comparative MapServer versus ArcGIS Server (ppt)
      Martine Girard
    • Étude comparative de deux systèmes de cache : GeoWebCache vs. TileCache (pptx)
      Élodie Edoh-Alove
  • 2008
    • Comparatif de deux implémentations open source de services web OGC : MapGuide OpenSource et Geoserver (ppt)
      Sandrine Émonin
    • Comparaison des outils open source de webmapping Mapfish et Mapbuilder (ppt)
      Fabien Ancelin
    • Étude des formats de publication de données spatiales sur le web : X3D et Collada (ppt)
      Belko Diallo
